## Anticipated cancer-genomics highlights *(expected, program not yet public)*

Cross-cutting prediction for the cancer-genomics subset we care about most:

### 1. Polygenic risk scores (PRS) in oncology — cross-ancestry transferability
- Expected updates from Mavaddat / Pharoah-aligned consortia on **breast-cancer PRS** transportability and clinical integration
- **Prostate-cancer PRS** updates anchored on UK Biobank + MVP; potential clinical-utility readouts that seed downstream ASCO trial-stratification work
- **Colorectal / pancreatic PRS** — emerging cohorts, particularly cross-ancestry validation
- Methods: PRS-CSx, SBayesRC, BridgePRS, and successors

### 2. Hereditary-cancer panel design — VUS classification at population scale
- **gnomAD v5** release (expected at ASHG) will re-baseline allele frequencies, potentially reclassifying VUSes in BRCA1/2, MMR, ATM, PALB2, CHEK2 etc.
- **Universal-testing-for-cancer** evidence — increasing call for tumor-agnostic germline testing in newly-diagnosed cancers; ASHG is where the prevalence data drops first
- **Variant interpretation automation** — foundation-model variant-effect predictors (AlphaMissense, EVE, ESM-3 successors) at clinical decision-point

### 3. Somatic mosaicism and clonal hematopoiesis (CHIP/CCUS)
- Population-scale **CHIP prevalence** updates from All of Us, UKB, MVP — feeds into ASH 2026 MDS/AML pre-disposition talks
- **MN-predisposition syndromes** (DDX41, RUNX1, GATA2) variant-classification updates
- Methods: deepMosaic, MosaicHunter, MosaicForecast updates; UMI-error-corrected sequencing applications

### 4. Large-cohort sequencing — All of Us + UK Biobank cancer-genomics co-readouts
- **All of Us v8** release with pan-cancer germline-somatic overlay (expected)
- **UK Biobank 500k WGS** mature data — cancer-incidence-linked germline analyses
- **Million Veteran Program** cancer-genomics module updates
- Coordinated cross-biobank meta-analyses via **GBMI** (Global Biobank Meta-analysis Initiative)

### 5. Foundation-model variant effect predictors for cancer
- **AlphaMissense** follow-on methods; cancer-specific training-set fine-tuning
- DNA-language models (Evo / Caduceus / Genome-LM successors) applied to non-coding cancer variants
- Cross-link target: AACR 2026 ML-bio sessions, ISMB 2026 deep-learning track
