## Track shortlist — most-loaded tracks

ASTRO 2026 will run ~2,500 abstracts across proffered papers, posters, and digital posters. The following tracks are expected to dominate by volume and discussant attention:

### GU Oncology (largest single-disease track)
- **Prostate:** NRG-GU005 long-term; STAMPEDE-2 RT arm; PSMA-directed re-irradiation; oligo-progression SBRT/SABR; brachytherapy + EBRT (ASCENDE-RT 10-yr); post-prostatectomy POSTILV / SAKK; HYPRO long-term hypofractionation.
- **Bladder:** BART follow-up (post-op RT after radical cystectomy); bladder-preservation chemo-RT update.
- **GYN:** Brachytherapy outcomes (CC + EBRT), interstitial implants; vaginal-cuff hypofractionation.

### Lung
- **Early-stage NSCLC SBRT:** RTOG-F successor to 0813 / 0915 fractionation studies; pre-op SBRT + IO neoadjuvant; SBRT-vs-surgery (STARS / ROSEL-type designs).
- **Locally advanced NSCLC:** PACIFIC-9 / successor (chemo-RT + durva + add-on); proton thoracic (RADCOMP / PROTECT-NSCLC analogue).
- **SCLC:** NRG-LU007 TRT fractionation; cross-link to ASCO 2026 TRIPLEX LBA8005 + ESMO 2026 SCLC slate.

### Breast
- **Partial-breast accelerated (APBI):** FAST-FORWARD partial-breast update.
- **Regional-nodal hypofractionation:** RT CHARM follow-up.
- **Skip-RT after neoadjuvant pCR:** NSABP B-51 / NRG-RT-005 — major SABCS 2026 dual-presenter candidate.
- **Proton breast:** RADCOMP cardiac follow-up Plenary candidate.
- **Single-fx WBI:** EMPOWER or analogue.

### CNS / Brain mets
- **SRS fractionation, post-op cavity SRS, leptomeningeal:** NRG-CC008 hippocampal-sparing + memantine vs SRS for >10 mets.
- **Pediatric proton CSI long-term:** registry / cooperative-group readouts.
- **Recurrent glioma re-irradiation:** registry data + early-phase.

### GI
- **Pancreatic SBRT (MR-linac adaptive):** ARTNET consortium readouts; LAP07 / Alliance follow-up.
- **Anal canal de-escalation:** PLATO follow-up.
- **Rectal TNT non-operative:** OPRA / RAPIDO long-term.
- **Esophageal proton:** PROTECT.

### Head & Neck
- **HPV+ de-escalation:** NRG-HN005 follow-up, MC1675 / MC1273.
- **Proton H&N:** TORPEdO long-term + subgroup.
- **Re-irradiation registry:** multi-institution data.

### Heme
- **Hodgkin / MALT / orbital / CNS lymphoma:** field-size and dose-de-escalation.
- **CAR-T bridging RT:** small registries / cooperative reports.

### Technology / Physics — the AI-tools secondary track
- **MR-linac adaptive (Unity / MRIdian):** ARTNET + vendor consortium readouts. Online-adaptive workflow validation.
- **FLASH-RT:** FAST-02 expansion; preclinical-to-clinical translation. Vendor-led (IntraOp, Varian FLASHforward, Mevion).
- **Proton arc therapy:** early clinical implementation.
- **Carbon-ion:** NIRS / Heidelberg / Shanghai outcomes; emerging US sites.
- **AI in treatment planning / contouring:** auto-contouring validation (Limbus AI, MIM, Siemens AI-Rad Companion); synthetic-CT for MR-only workflows; dose-prediction CNNs / KBP automation; Varian Ethos Intelligent Optimization; Elekta Unity online-adaptive ML.

### Immuno-RT
- RT + IO mechanism + early-phase trials (NRG-LU007, FORCE, WITH-RT analogues); abscopal-effect biomarkers; cross-link to AACR 2026 radiation-immunology mechanism sessions.

## AI-tools secondary track — cross-link strategy

ASTRO has a meaningful AI-in-planning track (~50–150 abstracts annually) but the meeting is fundamentally clinical. The strategy for this vault:

- **Inside each trial page**, log named AI tools in the `## Radiation specifics → AI tooling used` field and link out to the appropriate RSNA 2026 / MICCAI 2026 tool dossier. Don't duplicate the dossier.
- **In `trials/index.md`**, the AI-tools cross-link table aggregates which tools appear in which trials.
- **If volume warrants** (>10 standalone tool entries post-meeting), promote to a `tools/index.md` directory and follow the RSNA per-tool template.

Key cross-link targets at the methods / imaging-FM siblings:

| Tool family | Where the dossier lives |
|---|---|
| Auto-contouring (OARs, target volumes) | [MICCAI 2026 tools](../miccai-2026/) (methods) + [RSNA 2026 tools](../rsna-2026/tools/index.md) (clinical-deployment) |
| Synthetic-CT (MR-only workflows) | [MICCAI 2026 tools](../miccai-2026/) (GANs / diffusion methods) |
| Dose-prediction CNNs / KBP | [MICCAI 2026 tools](../miccai-2026/) |
| Vendor adaptive workflows (Ethos, Unity) | [JPM 2026](../jpm-2026/index.md) (commercial / financial layer) + here when trial-named |
| Pathology / radiology FMs adapted for contouring | [AACR 2026 bioinfo-tools](../aacr-2026/topics/bioinfo-tools/) + [RSNA 2026 tools](../rsna-2026/tools/index.md) |

## Cooperative-group calendar (for trial-shortlist tracking)

ASTRO trials disproportionately originate from cooperative groups. Track these public calendars for what's reading out at ASTRO 2026:

- **NRG Oncology** — quarterly investigator calls; published trial pipeline at [nrgoncology.org](https://www.nrgoncology.org/). NRG-led trials at ASTRO usually preview ~3 months pre-meeting.
- **RTOG Foundation** — pipeline page; works closely with NRG and industry partners.
- **EORTC Radiation Oncology Group** — European cooperative; some trials dual-present at ASTRO + ESMO.
- **CCTG (Canadian)** — SABR-COMET lineage origins.
- **CRUK** — UK trials (TORPEdO, FAST-FORWARD lineage).
- **SAKK (Swiss), SWOG, Alliance, ECOG-ACRIN** — site-specific cooperative groups whose RT-arm trials surface at ASTRO.
- **NCI-CTEP** — multi-institutional NCI-funded studies (esp. immuno-RT / mechanism).

- **Now (May 11, 2026, T-4.5 months):** vault scaffolded (index.md, trials/index.md, this file). Per-trial shortlist seeded from cooperative-group calendars + 2025-carryover topline candidates.
