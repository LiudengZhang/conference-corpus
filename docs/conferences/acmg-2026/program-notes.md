## Session-shape inventory (verified or expected)

### Plenary slots

- **Presidential Plenary — *The Future of Expanded Newborn Genomic Screening*** (Wed Mar 11). Combined with ACMG Foundation Awards ceremony and David L. Rimoin Lifetime Achievement Award Lecture (Carey).
- **Additional named-award lectures** — distributed across plenary mornings (Watson, King, Foundation awards).

### Featured symposia / special sessions (verified per pre-meeting press)

1. **Non-Invasive Fetal Sequencing (NIFS) Is Coming: Are You Ready?** — prenatal screening expansion beyond NIPT to whole-fetal sequencing.
2. **Clonal Hematopoiesis, Somatic and Post-Zygotic Mosaicism in Multigene Cancer Testing for Geneticists** — clinical-lab translation of CHIP / mosaicism findings.
3. **Beyond The List — Scope and Implementation of ACMG Secondary Findings** — debate on SF list expansion (current v3.3) and pediatric reporting.
4. **Diagnostic challenges from the Undiagnosed Diseases Network (UDN)** — rare-disease diagnostic odyssey case readouts.
5. **Washington policy session** — FDA LDT final-rule implications, CMS coverage, payer dynamics.
6. **Genomic Oncology track** — hereditary cancer, germline-somatic integration, pharmacogenomics in oncology.
7. **Gene Therapy implementation** — clinical-lab perspective on AAV / CRISPR / antisense-oligonucleotide therapeutic deployment.

## Tracks and expected corpus weighting

| Track | Cancer-genomics relevance | Notes |
|---|---|---|
| **Cancer Genetics** | Core | Hereditary cancer (HBOC, Lynch, FAP, Li-Fraumeni, PJS, Cowden, MEN, vHL, RB), germline-somatic integration, CHIP / mosaicism in cancer testing, ACMG SF cancer subset, universal-testing implementations. |
| **ACMG/AMP Variant Classification Standards** | Core | Underpins all clinical cancer-variant calling. SVI evidence-recalibration, ClinGen VCEP-specific specifications, AlphaMissense / LLM-as-PP3 controversies. |
| **AI in Clinical Genomics** | Core | LLM reproducibility benchmark (verified platform talk), AI-assisted genome interpretation pipelines, foundation-model variant-effect predictors in clinical workflows. |
| **Prenatal / Perinatal / Reproductive Genetics** | Indirect | NIFS, expanded NIPT, expanded carrier screening, fetal exome/genome. Pediatric-cancer-predisposition reporting from fetal/NBS sequencing is the cancer-relevant axis. |
| **Newborn Screening (NBS)** | Indirect (Presidential Plenary core) | Expansion-to-genomic-sequencing debate; BabySeq2, GUARDIAN, NBSeq, EarlyCheck. Pediatric cancer-predisposition reporting wedge. |
| **Cardiovascular Genetics** | Adjacent | HCM / DCM / LQTS panels — methods overlap with cancer panels, not biology. |
| **Rare-Disease Diagnostics** | Adjacent | UDN, GREGoR (presented at ASHG), WGS-first programs, long-read / RNA-seq for variant interpretation. Cancer-predisposition rare variants overlap. |
| **Biochemical / Metabolic Genetics** | Indirect | Inborn errors of metabolism — treatment focus. |
| **Therapeutics / Gene Therapy** | Adjacent | AAV / CRISPR / ASO therapy clinical-lab implementation. RB1 / TP53 gene-therapy intersections. |
| **Health Services / Implementation / ELSI** | Indirect | Payer / FDA / LDT framework — same framework gates cancer-LDT operations. |
| **Education / Training / Workforce** | Indirect | Genetic-counseling workforce, residency/fellowship curricula. |
