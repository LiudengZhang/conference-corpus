# Trials — ESMO 2026

The primary artifact of this vault: **one Markdown file per trial / readout**. ESMO 2026 is a clinical-readout meeting (the European counterpart to ASCO), so the unit of analysis is the trial — not the speaker, not the tool, not the company.

> **Status:** Template + master index, meeting **5 months out** (as of May 11, 2026). Per-trial pages will be populated as embargoes lift — ESMO releases accepted abstracts on/around the day of presentation, with LBAs day-of. The official ESMO 2026 program won't be public until **~July/August 2026**, so the table below is a *very tentative* watch list seeded from H2 2026 phase-3 readout timing previews + ASCO 2026 follow-up candidates. Everything is flagged `*expected, program not yet public*` until the ESMO Online Programme posts. Numbers can — and will — shift on the actual abstract reveal.

## Per-trial template

Each trial file (e.g., `harmoni-6-os.md`) follows this schema. Copy and fill:

```markdown
# <Trial acronym / name> — <Drug regimen, indication, line>

> **Abstract:** <abstract #, e.g., LBA1> · **Session:** <Presidential Symposium I / Proffered Paper / Mini-Oral, date, time CEST, hall> · **Embargo:** <date>

## At a glance

- **Sponsor:** <pharma / cooperative group — note EORTC / GETUG / GBG / AGITG / GERCOR / ICORG / academic sponsor if applicable>
- **PI / presenter:** <name, institution>
- **NCT ID / EudraCT:** <NCTxxxxxxxx / 20xx-xxxxxx-xx>
- **Phase:** <1 / 2 / 3 / 2-3>
- **Design:** <arms, randomization, blinding>
- **N:** <enrolled / analyzed>
- **Primary endpoint(s):** <PFS / OS / EFS / DFS / ORR / pCR / iDFS>
- **Comparator:** <SOC / placebo / active comparator>
- **Indication:** <tumor type, subtype, biomarker selection>
- **Line:** <neoadjuvant / adjuvant / 1L / 2L / 3L+ / perioperative>
- **Regulatory framing:** <EMA / FDA / both — sized for which filing>

## Headline result

<One paragraph. Numbers + HR + p-value if known. If pre-embargo, mark
`*expected per <company> topline <date>*` or `*pre-embargo per <analyst source>*`.>

## Mechanism / class

<Drug class, target, prior approvals if any. Cross-link to related AACR
mechanism sessions and ASCO 2026 readouts.>

## Discussant takeaway

<Named discussant + key framing once known. Discussant identity is itself a
signal — ESMO assigns senior European figures to Presidential Symposium LBAs,
and the discussant framing typically defines the EU narrative.>

## Practice-changing?

<Yes / no / pending, with rationale: ESMO Clinical Practice Guideline implications,
EMA filing path, NCCN read-through, prior SOC.>

## Cross-links

- **ASCO 2026:** <related Plenary / oral / abstract — frequent, since ESMO often follows ASCO with OS / longer follow-up>
- **AACR 2026:** <related plenary / session / poster, if applicable>
- **AACR 2025:** <prior-year mechanism work>
- **Other ESMO 2026 trials:** <same class, same indication>
- **JPM 2026:** <sponsor presentation / deal>
- **Tools:** <if a biomarker / ctDNA / spatial method anchored the trial>

## Sources

- Annals of Oncology Abstract Book entry / DOI: <URL or "pending embargo">
- OncologyPRO session page: <URL>
- Company press release: <URL>
- Trade-press recap: <OncLive / Targeted Oncology / ESMO Daily Reporter URL>
- ClinicalTrials.gov / EudraCT: <NCT / EudraCT URL>
```

## Cross-cutting themes (for `themes.md`)

To synthesize across `trials/` once populated (provisional, will be reconciled post-meeting):

1. **OS confirmation of ASCO 2026 PFS winners.** HARMONi-6, Rasolute-302, frontMIND, ASCENT-04, SERENA-6, LIBRETTO-432 are the candidates. ESMO is often the "did the OS hold?" follow-up venue for trials that read PFS at ASCO.
2. **ADC sequencing becomes a clinical question.** SATEEN (TROP2 + HER2 post-Enhertu), Dato-DXd patient-reported follow-ups, novel-target ADCs entering phase 3. The clinical narrative shifts from "does the ADC work?" (settled) to "what comes after the ADC?".
3. **CDK4/6 post-progression endocrine continuation.** evERA giredestrant + everolimus and SERENA-6 PFS2 mature; positions next-gen oral SERDs vs the AI + CDK4/6 backbone.
4. **PD-1×VEGF bispecific class read-through expands.** HARMONi-6 OS confirmation, plus first phase-2 / phase-3 data from AbbVie RC148 (Summit-licensed) and BMS pumitamig. Continuation of the JPM 2026 "PD-1×VEGF class" thread.
5. **ctDNA-guided trial designs continue.** ESMO 2025 Presidential featured IMvigor011 (MIBC) and DYNAMIC-III (stage III CRC) — both will likely have follow-up. New entrants likely.
6. **Perioperative IO matures.** KEYNOTE-B15 (bladder), PROTEUS (prostate) OS / EFS follow-ups from ASCO 2026 readouts.
7. **EMA vs FDA filing geography.** Several ESMO 2026 trials are sized for EMA-first filings. Worth tagging in per-trial sheets.
8. **European cooperative-group trial volume.** EORTC, GETUG, GBG, AGITG, GERCOR — the ESMO-native sponsors. ASCO undercovers these; ESMO is their home venue.
