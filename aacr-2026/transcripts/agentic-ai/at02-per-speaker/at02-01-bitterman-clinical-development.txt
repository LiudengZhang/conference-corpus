Now I'm going to quickly go into an example of how my lab has been using agentic AI for one part of this workflow, which is monitoring of adverse events in real time from electronic health records.So a current bottleneck for real-world data abstraction as well as for clinical trial curation is that we currently rely largely on manual abstraction for a lot of the information that's documented in the EHR. To address this challenge, we developed an agentic AI system that takes in clinical notes and extracts adverse events, specifically immunotherapy-related adverse events, from those notes. It first determines whether an event is present in a clinical note, grades the severity of the notes according to CTCAE definitions, determines whether that adverse event is attributed to the immunotherapy, and determines the certainty with which that adverse event is attributed to the immunotherapy. This leads to a pipeline that goes from a fully unstructured clinical note that historically would have required human review to structured output in a JSON file that clearly delineates whether an IRE is present, the grade attribution, and the underlying clinical evidence for the attribution. Our system achieved a macro F1 of 0.93.
